# 主要功能
## 1.文档管理
新建文档：创建新的文档文件，支持多文档同时编辑。进入软件后点击"文件"菜单→"新建"选项或工具栏新建按钮，即可创建新的文档，文档新建成功后状态栏左侧会有提示。支持同时打开多个文档进行编辑，各文档独立显示在编辑区域。新建文档采用标准A4页面尺寸，默认应用标准字体和页面边距设置。

![image](https://github.com/Kintaro-Nino/Qt-Project/blob/WordOffice/images/image%20(20).png)

打开文档：打开现有的HTML格式文档，自动识别文本内容。进入软件后点击"文件"菜单→"打开"选项或工具栏打开按钮，选择需要打开的HTML格式文档或其他文档。软件自动识别文档内容格式，支持富文本和纯文本的自动识别与正确显示。可同时打开多个文档并在编辑区域进行切换操作。

保存文档：保存文档内容，支持直接保存和另存为功能。编辑完成后点击"文件"菜单→"保存"选项或工具栏保存按钮，即可保存当前文档内容，文档保存成功后状态栏左侧会有提示。首次保存时将弹出"另存为"对话框，可选择保存路径和文件名。支持将文档保存为HTML格式，自动保留所有文字格式和排版样式。已保存的文档再次编辑时可直接保存更新内容。


## 2.打印文档
点击"文件"菜单→"打印预览"选项，可预览文档打印效果，查看页面布局和内容排版情况。点击"文件"菜单→"打印"选项，在弹出的打印对话框中设置打印机参数，包括纸张大小、打印方向和打印范围等选项。



## 3.文字编辑
撤销重做：支持多步撤销和重做操作。点击"编辑"菜单→"撤销"选项或工具栏撤销按钮，可撤销上一步编辑操作。支持多步连续撤销，直至文档恢复到最初状态。点击"编辑"菜单→"重做"选项或工具栏重做按钮，可重新执行被撤销的操作。支持多步连续重做，恢复被撤销的编辑内容。操作过程中，软件会实时记录编辑历史，确保撤销重做功能准确执行。















剪切功能：选中需要剪切的文本内容，点击"编辑"菜单→"剪切"选项或工具栏剪切按钮，即可将选中内容移至剪贴板。支持快捷键Ctrl+X快速操作。


复制粘贴功能：选中需要复制的文本内容，点击"编辑"菜单→"复制"选项或工具栏复制按钮，即可将选中内容复制到剪贴板。支持快捷键Ctrl+C快速操作。在目标位置点击"编辑"菜单→"粘贴"选项或工具栏粘贴按钮，即可将剪贴板中的内容粘贴到当前位置。支持快捷键Ctrl+V快速操作。



## 4.格式设置
字体样式：设置文字加粗、倾斜、下划线等样式。加粗设置：选中需要设置的文本，点击"格式"菜单→"字体"→"加粗"选项或工具栏加粗按钮，即可将文字设置为粗体显示。倾斜设置：选中需要设置的文本，点击"格式"菜单→"字体"→"倾斜"选项或工具栏倾斜按钮，即可将文字设置为斜体显示。下划线设置：选中需要设置的文本，点击"格式"菜单→"字体"→"下划线"选项或工具栏下划线按钮，即可为文字添加下划线。







字体颜色：自定义文字颜色，支持颜色选择器。颜色设置：选中需要设置的文本，点击"格式"菜单→"字体颜色"选项或工具栏颜色按钮，弹出颜色选择对话框，在对话框中可选择预设颜色或自定义颜色值。


字体设置：设置字体家族和字体大小。字体：点击工具栏字体家族下拉框，可查看系统所有可用字体。选中文本后选择相应字体，即可更改文字字体样式，实时预览显示效果。字体大小：点击工具栏字体大小下拉框，可选择标准字号选中文本后调整字号，文字大小即时更新显示。



段落对齐：提供左对齐、居中对齐、右对齐、两端对齐。左对齐：选中需要设置的段落，点击"格式"菜单→"段落"→"左对齐"选项或工具栏左对齐按钮，即可将段落文本沿左边界对齐显示。居中对齐：选中需要设置的段落，点击"格式"菜单→"段落"→"居中对齐"选项或工具栏居中对齐按钮，即可将段落文本在页面居中显示。右对齐：选中需要设置的段落，点击"格式"菜单→"段落"→"右对齐"选项或工具栏右对齐按钮，即可将段落文本沿右边界对齐显示。两端对齐：选中需要设置的段落，点击"格式"菜单→"段落"→"两端对齐"选项或工具栏两端对齐按钮，即可将段落文本同时沿左右边界对齐显示，自动调整字间距使文本整齐排列。






## 5.页面设置
添加页面：在文档中添加新的空白页面。点击"窗口"菜单→"添加空白草稿页"选项，即可在当前文档末尾添加一个新的空白编辑页面。支持连续添加多个空白页面，满足文档编辑需求。添加成功后状态栏会显示操作提示信息。



删除页面：删除空白或多余的页面。点击"窗口"菜单→"删除空白草稿页"选项，系统将自动检测并删除文档中所有内容为空的空白页面。确保文档始终保留至少一个编辑页面，防止误删导致无页面可用。删除操作执行后，状态栏会显示删除成功的提示信息。
